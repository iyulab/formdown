It combines the simplicity of Markdown with powerful form capabilities, allowing you to create complex forms with minimal syntax.</p>
<h2>Core Philosophy</h2>
<ul>
<li><strong>Human-readable</strong>: Forms should be readable and writable by humans</li>
<li><strong>Simple syntax</strong>: Minimal learning curve with intuitive field definitions</li>
<li><strong>Web standards</strong>: Generates clean, semantic HTML forms</li>
<li><strong>Framework agnostic</strong>: Works with any web framework or vanilla JavaScript</li>
</ul>
<h2>Architecture</h2>
<p>Formdown consists of three main packages:</p>
<h3>@formdown/core</h3>
<p>The parsing engine that converts Formdown syntax into structured data.</p>
<h3>@formdown/ui</h3>
<p>Web components and utilities for rendering forms in the browser. This is the main package most users will need.</p>
<h3>@formdown/editor</h3>
<p>Development tools including a live editor with syntax highlighting and real-time preview.</p>
<h2>Key Features</h2>
<ul>
<li><strong>Shorthand syntax</strong>: Write forms faster with intuitive shortcuts (<code>@name*: []</code>, <code>@email: @[]</code>)</li>
<li><strong>Smart label generation</strong>: Automatic conversion of field names to readable labels</li>
<li><strong>Inline fields</strong>: Embed form fields directly in markdown text</li>
<li><strong>Rich field types</strong>: Support for all HTML5 input types plus custom components</li>
<li><strong>Flexible attributes</strong>: Control validation, styling, and behavior</li>
<li><strong>Markdown integration</strong>: Mix forms with rich text content</li>
<li><strong>Progressive complexity</strong>: Start simple, add complexity as needed</li>
<li><strong>Zero dependencies</strong>: Lightweight and fast</li>
</ul>
<h2>Example</h2>
<h3>Standard Syntax Example</h3>
<pre class="language-markup"><code class="language-markup">### Registration Form

@age: [number min=18 max=100]
@bio: [textarea rows=4 placeholder="Tell us about yourself"]
