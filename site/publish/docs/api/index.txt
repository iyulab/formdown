<h4>Radio Fields</h4>
<ul>
<li><strong>Single selection</strong>: Only one value can be selected</li>
<li><strong>Other option</strong>: Automatically used for values not in predefined options</li>
<li><strong>Clear</strong>: Deselects all options</li>
</ul>
<h4>Checkbox Fields</h4>
<ul>
<li><strong>Multiple selection</strong>: Multiple values can be selected simultaneously</li>
<li><strong>Other option</strong>: Can be used alongside predefined options</li>
<li><strong>Add/Remove</strong>: Supports granular value management</li>
</ul>
<h4>Select Fields</h4>
<ul>
<li><strong>Single selection</strong>: Only one value can be selected</li>
<li><strong>Other option</strong>: Shows text input when selected</li>
<li><strong>Clear</strong>: Resets to empty selection</li>
</ul>
<h4>Text/Textarea Fields</h4>
<ul>
<li><strong>Direct value</strong>: Set and get text values directly</li>
<li><strong>No other option</strong>: Not applicable to text fields</li>
</ul>
