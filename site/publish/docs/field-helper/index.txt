<h3>Radio Fields</h3>
<ul>
<li><strong>Selection</strong>: Single value only</li>
<li><strong>Other option</strong>: Automatically used for non-existing values</li>
<li><strong>Clear</strong>: Deselects all options</li>
</ul>
<h3>Checkbox Fields</h3>
<ul>
<li><strong>Selection</strong>: Multiple values supported</li>
<li><strong>Other option</strong>: Can be used alongside existing options</li>
<li><strong>Management</strong>: Granular add/remove/toggle operations</li>
</ul>
<h3>Select Fields</h3>
<ul>
<li><strong>Selection</strong>: Single value only  </li>
<li><strong>Other option</strong>: Shows text input when selected</li>
<li><strong>Clear</strong>: Resets to empty selection</li>
</ul>
<h3>Text/Textarea Fields</h3>
<ul>
<li><strong>Value</strong>: Direct string assignment</li>
<li><strong>Other option</strong>: Not applicable</li>
</ul>
